# 🖱️ MouseController

Удобное мобильное приложение на Android для управления курсором мыши на ПК через домашнюю локальную сеть (Wi-Fi). Превращает экран вашего смартфона в полноценный тачпад.
Проект написан с помощью ИИ

## ✨ Особенности проекта
* **Тачпад:** Плавное перемещение курсора пальцем по экрану смартфона.
* **Клик по тапу:** Быстрое касание экрана считывается как клик ЛКМ (защищено от случайных срабатываний во время движения).
* **Физические кнопки:** Выделенные кнопки для ЛКМ и ПКМ внизу экрана.

* **Удобные настройки:** Быстрое изменение IP-адреса компьютера и чувствительности мыши через всплывающее диалоговое окно на смартфоне.

## 📦 Установка и запуск

Для работы системы вам понадобится установить клиент на смартфон и запустить фоновый сервер на компьютере. Они должны находиться в одной сети.

### 📱 1. Установка на Android
1. Скачайте актуальный файл `app-debug.apk` из раздела [Releases (Релизы)](../../releases) проекта.
2. Перенесите файл на смартфон и запустите его установку.
   > **Примечание:** Если Android выдаст предупреждение об установке из неизвестных источников, разрешите установку (это стандартная защита для приложений, собранных вручную).
3. Запустите приложение **MouseController** на телефоне.

### 💻 2. Установка сервера на ПК (Windows)
1. Скачайте актуальный файл `MouseController.exe` из раздела [Releases (Релизы)](../../releases) проекта.
2. Запустите исполняемый файл `MouseController.exe`.
3. Программа мгновенно свернется в **системный трей** (скрытые значки возле часов) в виде зеленого круглого индикатора.
4. Можно включить автозапуск и изменить его в диспетчере задач.

#### Запуск

1. Запустите приложение `MouseController.exe` на ПК 
2. Запустите приложение `app-debug.apk` на смартфоне
3. В настройках приложения `app-debug.apk` укажите P-адрес компьютера
    Узнать его можно прямо в ПК:
    Откройте командную строку Windows (Win + R - введите cmd - Enter).
    Наберите команду ipconfig и нажмите Enter.
    Найдите строку IPv4-адрес (например, 192.168.1.50).

## ⚙️ Как это работает
Связь между устройствами реализована через быстрый сетевой протокол **UDP** на порту `5005`. 
1. Смартфон фиксирует смещение пальца на тачпаде, умножает его на выбранную чувствительность и отправляет строку вида `X,Y` на IP-адрес компьютера.
2. При одиночном тапе или нажатии кнопок отправляются текстовые команды: `click` или `r_click`.
3. Python-сервер на ПК слушает порт, мгновенно парсит входящие пакеты и перемещает курсор с помощью библиотеки `pyautogui`.
